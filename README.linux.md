## Installation

tl;dr:

    git clone https://github.com/seirl/tsmapplication_public
    cd tsmapplication_public
    git checkout linux

    python3 -m venv .venv
    source .venv/bin/activate
    pip install -r requirements-linux.txt

    ./get_private_config.sh
